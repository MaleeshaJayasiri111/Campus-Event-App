Campus Event Management System (PWA)

A Progressive Web App (PWA) built with **Blazor WebAssembly** and **Firebase** to manage and track campus events efficiently.

 🚀 Features
- Real-time Event Browsing: View upcoming campus events with full details.
- RSVP Tracking:Students can register their interest for events.
- Feedback System: Users can submit feedback on past events.
- Admin Dashboard: Secure management for event creators to add, edit, or delete events.
- PWA Support: Installable on mobile and desktop for an app-like experience.

🛠️ Tech Stack
- Frontend: Blazor WebAssembly (.NET)
- Backend/Database: Firebase Firestore
- Authentication:Firebase Auth
- Styling: Bootstrap 5 & Custom CSS

📱 Mobile-First Testing
This system has been rigorously tested using a mobile-first strategy to ensure full responsiveness and functionality on all mobile devices.

 🔧 How to Run Locally
1. Clone the repository: `git clone [https://github.com/MaleeshaJayasiri111/Campus-Event-App.git]`
2. Open the `.sln` file in Visual Studio 2022.
3. Ensure you have the `.NET 6/7/8 SDK` installed.
4. Press `F5` to run the application.
